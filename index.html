Configuración inicial — solo una vez
✅ Solo necesitas crear una cuenta gratis en Firebase (Google). Tarda 5 minutos.
1
Abre Firebase Console
Ve a console.firebase.google.com e inicia sesión con Gmail.
2
Crea un proyecto
Clic en Agregar proyecto → Nombre: a-brasas → Desactiva Google Analytics → Crear proyecto.
3
Activa Realtime Database
En el menú izquierdo: Compilación → Realtime Database → Crear base de datos → Elige la ubicación más cercana → Empezar en modo de prueba → Habilitar.
4
Copia la URL de la base de datos
Aparece en la parte superior, con este formato:
https://a-brasas-xxxxx-default-rtdb.firebaseio.com
5
Reglas de seguridad (modo prueba, 30 días)
El modo de prueba permite acceso libre 30 días. Suficiente para empezar. Puedes ajustar las reglas después si necesitas más tiempo.
